Definition

A cycloid is the curve traced out by a point on a circle as it rolls along a flat surface. Above, animating the graph will show the point on the wheel as the wheel rolls along the x-axis. If you increase the maximum for t, then you can make it go further than a single rotation. What happens if we do two full rotations? Adjust the maximum to 4π and animate.

Derivation of Equations

Where do the equations come from? First consider the circle with radius 1.

First, pay attention to the point P as it goes around the circle. It starts at the bottom of the circle and moves in the clockwise direction. In particular, the x-coordinate starts at 0, goes to the left (negative) and then to the right (positive), returning to 0. This matches the equation x(t) = -sin(t) y-coordinate starts at the minimum value, which matches with y(t) = -cos(t).

So the parametric equations for a point starting at the bottom of a circle and moving clockwise are:

x(t) = -sin(t)

y(t) = -cos(t)

But, we can see that the circle is also moving. First, we need to translate it up one unit by adding 1 to y(t). Also, it needs to be translated horizontally by a variable amount. To see how the translation relates the circle, click the checkbox for "Show Circumference" and watch how the circumference of the circle relates to the horizontal translation. When the point on the wheel has returned to the bottom (when t=2π), the circle has completed one full rotation. Horizontally, the center has moved 2π units to the right. This is the same as the value of t, so we just need to add t to x(t).

So the equations for the cycloid when the radius is 1 are:

x(t) = -sin(t) + t

y(t) = -cos(t) + 1

If we change the radius to r in general, then we would need to adjust our equations for the circle by adding a coefficient of r to each of the trigonometric functions. For the translations, now we need to translate the circle up by r units. Horizontally, the wheel will complete one full rotation when the horizontal distance it has moved is the same as the circumference, or 2πr. This will still happen when t=2π, so we want a horizontal translation now of rt. Our general equations are:

x(t) = -rsin(t) + rt

y(t) = rcos(t) + r
